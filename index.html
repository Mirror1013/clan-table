⭐ Уважаемые клановые тренеры и руководители кланов!
На наших серверах работают новые админ-команды. Клановые администраторы имеют доступ к некоторым из них.

🎥 Инвиз

//spectatormode или //sp— инвиз по команде, можно выходить даже в воздухе

🩵 Хп

//sethealth [никнейм] [здоровье]— поставить игроку хп (по нику)
//sethealthid [SteamID] [здоровье]— поставить игроку хп (по стимайди)Если не указать здоровье — будет максимальное

🥩 Голод

//sethunger [никнейм] [голод]— поставить игроку голод (по нику)
//sethungerid [SteamID] [голод]— поставить игроку голод (по стимайди)Если не указать голод — будет максимальный

💦 Жажда

//setthirst [никнейм] [жажда]— поставить игроку жажду (по нику)
//setthirstid [SteamID] [жажда]— поставить игроку жажду (по стимайди)Если не указать жажду — будет максимальная

🩸 Блид

//setbleed [никнейм] [блид]— поставить игроку блид (по нику)
//setbleedid [SteamID] [блид]— поставить игроку блид (по стимайди)Если не указать блид — будет 0

🦴 Перелом

//setlegdamage [никнейм] [перелом]— поставить перелом лапы (по нику)
//setlegdamageid [SteamID] [перелом]— поставить перелом лапы (по стимайди)Максимальный перелом — 15. Если не указать — 0

❣️ Хил

//selfheal— отхилить себя
//heal [никнейм]— отхилить игрока (по нику)
//healid [SteamID]— отхилить игрока (по стимайди)

🦕 Гров

//selfgrowth [рост]— установить рост для себя
//setgrowth [никнейм] [рост]— установить рост игроку (по нику)
//setgrowthid [SteamID] [рост]— установить рост игроку (по стимайди)Если не указать рост — максимальный. Минимальный рост у разных динозавров разный!

💀 Слэй

//selfslay [0/1] или //ss— слейнуть себя (0 — без трупа, 1 — с трупом)
//slay [никнейм] [0/1]— слейнуть игрока (по нику)
//slayid [SteamID] [0/1]— слейнуть игрока (по стимайди)По умолчанию 0

▶️ Тп к другому

//goto [никнейм]— тп к игроку даже в инвизе (по нику)
//gotoid [SteamID]— тп к игроку даже в инвизе (по стимайди)

◀️ Тп к себе

//bring [никнейм]— тп игрока к себе даже в инвизе (по нику)
//bringid [SteamID]— тп игрока к себе даже в инвизе (по стимайди)
